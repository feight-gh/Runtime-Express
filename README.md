Runtime Express —— 大量软件的运行平台的安装器
===============
这是一个运行库集成安装器。其中包含了以下的运行库：

>VC++ 2005 2008 2010 2012 2013 x86 & x64

>.net Framework 4.5.2

>DirectX 9c

>Microsoft Game For Windows - Live

>Microsoft WSE 3.0

>OpenAL 2.0.7

>PhysX 9.12.1031

>Microsoft XNA 2.0 3.1 4.0

>Java 8 Update 40 x86 & x64
