什么是 Runtime Express？

Runtime Express 是一个实用的集成环境安装器。通过它，你可以便捷地安装你所需要的运行库，让你的程序正常运行！

它有何优点？

更加聪明

Runtime Express 会根据你计算机的实际情况，选择恰当的安装包——当然，最终安装什么由你决定。

更加自由

Runtime Express 会给你充分的自由发挥空间，你可以调整是否静默安装，是否校验（尽管我们十分建议校验，因为这更安全）。

更加美观

得益于WPF，Runtime Express 拥有一个美观的界面。它也许不是最好看的，但它确实很美，而且也很安全。
